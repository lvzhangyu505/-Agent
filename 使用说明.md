# 使用说明

## 第一次使用

运行：

```bash
/Users/lvzhangyu/.cache/codex-runtimes/codex-primary-runtime/dependencies/python/bin/python3 biaoshu_agent.py init
```

然后把文件放进这些目录：

- `招标文件/`：本次项目招标文件。
- `企业资料/`：企业基础信息、法人信息。
- `资质证书/`：营业执照、许可证、资质说明等，建议放文字版或 docx。
- `企业业绩/`：历史项目业绩、合同摘要、案例说明。
- `历史标书/`：以前写过的标书，用于复用方案段落。
- `模板/`：可选 Word 模板。

## 一键生成

```bash
/Users/lvzhangyu/.cache/codex-runtimes/codex-primary-runtime/dependencies/python/bin/python3 biaoshu_agent.py run-all "招标文件/xxx.docx"
```

## 输出结果

在 `输出标书/项目名称/` 下查看：

- 招标文件解读
- 投标目录
- 标书初稿
- 合规检查报告
- 完整标书 Word

## 建议工作流

1. 先运行一键生成。
2. 打开 `04_合规检查报告.md` 看高风险问题。
3. 修改 `03_标书初稿.md`。
4. 再运行 `export` 导出 Word。
5. 在 Word 中检查页眉页脚、目录页码、签章和最终排版。
