标书生成工作台

上传招标文件，自动完成解读、目录、初稿、合规检查和 Word 导出。

支持 DOC、DOCX、PDF、TXT、MD。建议优先使用 Word 或可复制文字的 PDF。

2生成投标目录
3生成正文初稿
4合规检查
5导出 Word

部署说明

当前是本地可运行版本，后续可推送到 GitHub 后部署到服务器或容器平台。

python3 web_app.py --host 0.0.0.0 --port 8787

生产部署建议加反向代理、HTTPS、访问密码，并把资料目录挂载为持久化磁盘。
